# Why?

In one word: **security**.

No other Operating System - in it's **`default` configuration** -
comes _close_ to the **`OpenBSD` track record** of security.

# What?

`OpenBSD` is a UNIX Operating System
that focusses
[**_proactive_ security**](https://www.openbsd.org/security.html)
and includes
[**integrated cryptography**](https://www.openbsd.org/crypto.html).

From the
[openbsd.org](https://www.openbsd.org)
site:

"_The `OpenBSD` project produces a **FREE**,
multi-platform
4.4 BSD-based UNIX-like operating system.
Our efforts emphasize portability, standardization, correctness,
proactive security and integrated cryptography._"
